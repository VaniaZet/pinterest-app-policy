# Privacy Policy

## 1. Who we are
Chubatovcan_Autoposter is an application developed by an individual developer for personal use.

## 2. Data we process
The app may process the minimum necessary Pinterest data to function, such as:
- OAuth access tokens (authorization tokens)
- Board identifiers (board IDs)
- Pin data provided by the user (title, description, link, image URL)

## 3. How we use data
Data is used only to:
- create and publish Pins to Pinterest
- (optionally) list boards to select where to publish

## 4. Data storage and sharing
- Access tokens are stored only by the app owner (for example, in local environment variables) and are not shared with third parties.
- The app does not sell, rent, or share personal data with third parties.
